Fashion-MNIST Perceptron vs MLP Neural Comparison

 Deep Learning | TensorFlow | ML Framework | scikit-learn | PCA Visualizer | Accuracy Report | MIT License 

 This project performs classification and analysis on the Fashion image dataset using:

A custom coded single-layer Perceptron

A neural network with MLPClassifier

Dimensionality reduction with PCA

Activation behavior plots using Matplotlib

Dataset used:
Fashion‑MNIST Dataset
Loaded via: fashion_mnist.load_data() from Keras in TensorFlow

Preprocessing: Image flattening (28×28 → 784) and normalization
Training on full feature space (784-D) and PCA projected 2-D space
Decision region visualization for model understanding
Classic ML vs Neural accuracy benchmarking
Activation function graphs: Sigmoid, Tanh, ReLU
Entire perceptron algorithm implemented using NumPy

Model / Technique	Role
Custom Perceptron	Manually implemented linear classifier
MLPClassifier	Multi-layer non-linear neural model
PCA 2-D	Visualization of high-dimensional image space
Step Activation	Binary neuron firing logic
Sigmoid / Tanh / ReLU	Activation response analysis


Training logs (iteration print output)
PCA Decision Boundary Map (2-D contour regions)
2-D dataset scatter plot labeled by category
Activation function graphs for neuron insight
